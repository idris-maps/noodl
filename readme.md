### Update data

```
$  npm run data
```

### Generate menu

```
$ npm run svg
```

#### Create pdf

```
$ npm run pdf
$ cd svgs/pdf
$ pdfunite 00.pdf 01.pdf 02.pdf 03.pdf 04.pdf 05.pdf 06.pdf 07.pdf 08.pdf 09.pdf 10.pdf 11.pdf 12.pdf 13.pdf noodl.pdf

```

```

```

